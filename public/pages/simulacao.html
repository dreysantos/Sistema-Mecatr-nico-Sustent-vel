🔌 Mapa de Conexões

Este diagrama mostra como todos os componentes estão conectados ao Arduino e entre si:

Sensores de Nível A0, A1, A2

Entradas analógicas para detectar níveis de água (baixo, médio, cheio)

Sensor ACS712 (Corrente) A3

Entrada analógica para medir a corrente elétrica do sistema em amperes

Sensor de Tensão (Bateria) A4

Entrada analógica para monitorar a tensão e carga da bateria 12V

Sensor de Tensão (Solar) A5

Entrada analógica para medir a tensão gerada pela placa solar

Relé (Controle da Bomba) D13

Saída digital que controla o relé 5V, que por sua vez aciona a bomba 12V

Display LCD I2C SDA: A4, SCL: A5

Comunicação I2C para exibir dados em tempo real no display LCD 16x2

Comunicação Serial (USB) TX: D1, RX: D0

Porta serial para enviar dados em formato JSON para o servidor web

Alimentação 5V, GND, Vin (12V)

Arduino recebe 5V via USB ou conversor Buck. Sistema recebe 12V da bateria
