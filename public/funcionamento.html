Sistema Mecatrônico Automatizado

Painel local de acompanhamento das leituras do sistema de controle de água, energia e alimentação solar.

Início Painel Leituras Relatórios Alertas LCD Funcionamento Componentes Simulação Diagnóstico Sobre

Como o sistema funciona

O Arduino processa sensores de água, corrente, bateria e placa solar. Com essas leituras, ele controla a bomba e envia os dados para o servidor web.

1. Leitura dos sensores

Os sensores identificam nível de água, corrente consumida, tensão da bateria e tensão da placa solar.

2. Processamento no Arduino

O Arduino interpreta os sinais e decide se a bomba deve ligar, desligar ou permanecer pausada.

3. Controle da bomba

O relé funciona como chave eletrônica para acionar a bomba de água com segurança.

4. Monitoramento energético

O sistema acompanha bateria, placa solar e consumo para evitar falhas de energia.

5. Exibição no site

As informações chegam ao servidor Node.js por USB serial e são exibidas em tempo real no navegador.

6. Alertas visuais

Condições como bateria baixa ou corrente alta aparecem com destaque no painel.

Fluxo de dados

Sensores -> Arduino -> Lógica de controle -> Relé/Bomba

JSON serial -> Servidor Node.js -> Navegador -> Painel em tempo real
