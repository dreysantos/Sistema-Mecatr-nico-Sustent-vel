ℹ️ Sobre o Projeto

O Sistema Mecatrônico Automatizado para Controle Inteligente de Água, Energia e Alimentação Solar foi desenvolvido para demonstrar como a automação e a sustentabilidade podem trabalhar juntas para economizar recursos naturais.

🎯 Objetivo do Projeto:

Criar um sistema inteligente que monitore e controle de forma autônoma o consumo de água, o uso de energia solar e a carga de bateria, demonstrando aplicações práticas de automação para sustentabilidade.

🌟 Destaques do Sistema

⏱️ Tempo Real

Monitoramento contínuo com atualização a cada segundo, permitindo decisões rápidas.

📱 Interface Web

Acesso via navegador em qualquer dispositivo conectado à rede local.

🔋 Sustentável

Utiliza energia solar como fonte principal, reduzindo dependência de eletricidade.

🚨 Alertas Inteligentes

Sistema de notificação visual para situações que requerem atenção imediata.

📊 Dados Precisos

Sensores de alta qualidade garantem leituras confiáveis e consistentes.

🔧 Extensível

Arquitetura aberta permite adição de novos sensores e funcionalidades.

🏫 Contexto Educacional

Este projeto foi desenvolvido no CETEP-RM (Centro Estadual de Educação Tecnológica em Petroquímica Resina Metaconcreto) em Camaçari, Bahia, como parte de um programa educacional focado em tecnologia, sustentabilidade e inovação.

O projeto demonstra conceitos importantes de engenharia mecatrônica, programação de microcontroladores e desenvolvimento web, preparando estudantes para desafios do mundo real.
